Proyecto 1

Sistema de Punto de Venta para Restaurantes

Desarrollé un sistema de punto de venta (POS) completo para restaurantes, funcional como Progressive Web App (PWA) instalable en dispositivos móviles y de escritorio. El sistema incluye gestión de pedidos en tiempo real con vista de cocina, administración de menú e inventario con control de ingredientes, módulo de gastos y proveedores, corte de caja con exportación a CSV, integración con base de datos en Supabase, y soporte offline mediante Service Worker. Construido con JavaScript vanilla, CSS personalizado y SQL, sin frameworks externos.
